## Quick Start

1. **Clone the repo**
   ```bash
   git clone https://github.com/<your-org>/useful-scripts.git
   cd useful-scripts
   ```

2. **Pick a script** from the table above and open its file.

3. **Run it** — most scripts are self-contained. Language-specific instructions (e.g. `python script.py`, `node script.js`, `bash script.sh`) are noted in each script's header.
