# Dynamic Media Strip

**Scroll any page. Instantly grab every image in sight.**  
Pinterest, Instagram, Google Images, Behance, news galleries, stock photo catalogs — if it has pictures, this strip catches them.

A zero-extension, live-updating sidebar that docks to your screen as soon as you paste it. Hover any thumbnail for a full-res zoom preview, click the save button to open the original in a new tab. It tracks lazy-loaded images and infinite scroll out of the box.

## Usage

1. Open **any website with images** in your browser.
2. Open DevTools (`F12` → **Console** tab).
3. Paste the script and hit **Enter**.
4. A red panel appears in the top-right corner. Just scroll — every image gets captured in real time.

Hover a thumbnail for a large preview. Click the 💾 button to open the full-resolution file.

## Works on

- Pinterest, Instagram, Google Images, Behance, Dribbble
- News article galleries, e-commerce product shots
- Any site that loads images via `<img>`, `<video>`, or CSS `background-image`

If the page has pixels, the strip finds them.
