# Facebook Auto Friend Request

**Growth-hack your network on autopilot.**  
One paste. One tab. Hundreds of connections.

Drop this into Facebook's console and watch it systematically click every "Add Friend" button on the page with randomized 4–9 second delays that feel eerily human — built to scale without tripping alarms.

## Usage

1. Open **Facebook.com** and go to a page with "Add Friend" buttons (suggestions, mutual friends, search results).
2. Open DevTools (`F12` → **Console** tab).
3. Paste the script and hit **Enter**.
4. Sit back. It clicks through every button with a random 4–9s pause between each.

The console logs each click and the live count so you know exactly where it stands.
